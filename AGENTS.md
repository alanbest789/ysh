# 迎胜汇企业官方网站 - 需求拆解文档

## 产品概述

- **产品类型**: 企业官方网站（影视文化公司品牌展示站）
- **场景类型**: <scene_type>prototype-app</scene_type>
- **目标用户**: 影视行业合作方、编剧人才、投资方、品牌客户及潜在合作伙伴
- **核心价值**: 以高端国风红金视觉呈现迎胜汇在剧本开发、影视制作、教育培训三大业务板块的综合实力，展示代表作品与全球化布局，建立品牌信任并引导商务合作
- **界面语言**: 中文（zh-CN）
- **主题偏好**: user_specified（红金国风配色，深色底+金色点缀的高端商务国风）
- **导航模式**: 锚点导航
- **导航布局**: Topbar（顶部固定导航栏）

## 页面布局建议

- **布局模式**: 单页纵向滚动 + 居中容器（max-width 约 1200-1440px），各板块间以国风装饰元素（祥云纹、金线分隔）做视觉区隔
- **视觉重心**: 首屏 Hero 区为品牌视觉重心（红金渐变背景 + 龙纹祥云装饰 + 主标题），后续板块以"标题+内容"的标准模块节奏递进
- **结果承载区**: 不适用（品牌展示型官网，无输入/生成类交互）
- **首屏设计要点**:
  - 背景使用用户提供的参考图 URL（https://aka.doubaocdn.com/s/Cbop1wjraq）作为底图，叠加红色渐变蒙版确保文字可读性
  - 顶部融入龙和祥云的 SVG/CSS 装饰元素，与背景图呼应
  - 导航栏初始透明，滚动后变为深红/金色实底（毛玻璃+渐变效果）

## 设计规范补充

### 配色方案（红金国风）

| 色值 | 用途 |
|------|------|
| `#8B0000` / `#A52A2A` | 主背景深红、导航栏滚动后底色 |
| `#C41E3A` / `#DC143C` | 强调红、按钮背景、渐变起始色 |
| `#D4AF37` / `#FFD700` | 金色主色、标题文字、图标、按钮描边 |
| `#F5E6C8` / `#F0E68C` | 浅金、辅助文字、装饰线条 |
| `#1A0A0A` / `#2D1810` | 深色底、卡片背景（深色国风底） |
| `#FFFFFF` / `#FAF5EB` | 正文文字、米白底（浅色区块用） |

### 装饰元素

- **龙纹**: 顶部导航栏两侧/背景用线稿龙纹 SVG，金色描边，低透明度
- **祥云**: 板块分隔处、卡片角落用祥云图案装饰，金色线条
- **回字纹/万字纹**: 卡片边框、按钮边框可用传统纹样装饰
- **水墨晕染**: 板块过渡处可用红色水墨晕染效果做视觉分隔

### 字体建议

- **标题**: 宋体/楷体/书法字体（如"思源宋体"、"站酷高端黑"等），体现国风气质
- **正文**: 无衬线字体（如"思源黑体"），保证可读性
- **数字/英文**: 可搭配衬线字体增强高端感

### 动效规范

- **平滑滚动**: 全站 `scroll-behavior: smooth`
- **入场动画**: 各板块滚动到视口时从下方淡入上移（Intersection Observer）
- **悬停效果**: 卡片/按钮 hover 时金色光晕、轻微上浮
- **数据计数**: 数据卡片数字滚动计数动画
- **无限滚动**: 代表作品海报行使用 CSS animation + transform 实现无缝循环

# UI 设计指南

## 1. 设计推导依据

- **参考意图**: Mood Reference —— 参考图提供红金国风影视公司的氛围、配色与龙祥云母题，不照搬具体版式；飞书链接仅作为剧本库跳转目标，不影响视觉设计。
- **核心情绪 / 应用类型**: 高端国风影视内容公司品牌官网，传递厚重、专业、东方美学与影视工业感。
- **独特记忆点**: 首屏深红底上叠加金色龙纹与祥云剪影，配合胶片/摄影机暗纹，形成"龙跃祥云、光影叙事"的品牌视觉锚点。

## 2. Art Direction

- **方向名**: 朱金影韵 · 国风商务
- **Design Style**: 东方奢华 + Editorial 经典排版 —— 以中国传统朱红与鎏金为主调，结合现代影视公司官网的大版面叙事，既有文化厚重感又不失商业专业度。
- **DNA 参数**: 圆角 subtle（`rounded-md`，卡片/按钮微圆）/ 阴影 layered（金粉微光感 + 深色投影）/ 间距 spacious（`gap-6` / `p-8`）/ 字体方向（标题衬线显国风，正文无衬线保可读）/ 装饰手法（祥云纹底纹、金线分隔、龙形剪影、胶片暗纹）
- **应用类型**: Landing —— 单页长滚动叙事，七大板块层层递进。

## 3. Color System

**色彩关系**: 朱红主色 + 鎏金强调色 + 墨黑文字 + 暖米白辅助底 + 暗红渐变背景层。
**配色设计理由**: 朱红承载品牌识别与CTA行动，鎏金用于标题高亮、图标、装饰线条与重要数据，暖米白与暗红渐变构建层次丰富的背景系统，墨黑确保正文高对比可读。
**主色推导**: 从中国传统宫廷朱红（约 hsl 355°）出发，降低明度提升饱和度获得品牌深红；金色从传统鎏金（约 hsl 42°）衍生，用于点缀与高光；整体色温偏暖，呼应东方影视的厚重与仪式感。
**使用比例**: 55% 中性（深红底 + 米白卡片）/ 35% 辅助（暗红渐变 + 金线装饰）/ 10% primary（朱红CTA + 品牌锚点）；金色为 accent，不用于正文大面积。

| 角色 | CSS 变量 | Tailwind Class | HSL 值 | 设计说明 |
|---|---|---|---|---|
| bg | `--background` | `bg-background` | hsl(355 55% 12%) | 页面主背景，深朱红近黑，营造影院暗场感 |
| card | `--card` | `bg-card` | hsl(30 30% 96%) | 暖米白卡片面，承载正文与数据 |
| text | `--foreground` | `text-foreground` | hsl(30 15% 10%) | 卡片上正文，墨黑偏暖 |
| textMuted | `--muted-foreground` | `text-muted-foreground` | hsl(30 8% 40%) | 辅助说明与元信息 |
| primary | `--primary` | `bg-primary` / `text-primary` | hsl(355 75% 42%) | 朱红主色，用于CTA按钮、品牌标识、关键高亮 |
| primaryForeground | `--primary-foreground` | `text-primary-foreground` | hsl(42 100% 92%) | 主色上的金色文字，形成红金对比 |
| accent | `--accent` | `bg-accent` | hsl(42 80% 62%) | 鎏金强调色，用于标题、图标、装饰线、数据数字 |
| accentForeground | `--accent-foreground` | `text-accent-foreground` | hsl(355 55% 12%) | 金色底上的深朱红文字 |
| border | `--border` | `border-border` | hsl(42 40% 70%) | 金色细边框，用于卡片描边与分隔线 |

**语义色提示**: 成功色 hsl(142 55% 38%)（深翠绿，与主色同低明度），三态：bg hsl(142 45% 92%) / border hsl(142 50% 65%) / text hsl(142 55% 30%)；警告色 hsl(38 90% 50%)（暖金橙，与金色系统衔接），三态：bg hsl(42 85% 92%) / border hsl(42 80% 65%) / text hsl(30 80% 35%)；错误色 hsl(355 70% 45%)（与 primary 同源偏亮），三态：bg hsl(355 60% 92%) / border hsl(355 65% 70%) / text hsl(355 75% 35%)。所有语义色饱和度与 primary 对齐 ±15%。

## 4. 字体与节奏

- **font-display**: Noto Serif SC —— 中文衬线，传递国风厚重与影视叙事的文学感，用于板块大标题、Hero 主标题。
- **font-body**: Noto Sans SC —— 中文无衬线，保证长文阅读清晰度与商务专业感，用于正文、说明、按钮文字。
- **字号**: H1 text-5xl md:text-6xl lg:text-7xl（Hero 标题，金色渐变）；H2 text-3xl md:text-4xl（板块标题）；H3 text-xl md:text-2xl（卡片标题）；body text-base；muted text-sm。
- **圆角**: 小（`rounded-md`，6-8px）—— 呼应传统器物的微圆边角，不过度软化，保持庄重感。

## 5. 全局布局契约

- **Reference Layout Use**: 按需求结构推导；参考图仅影响首屏氛围与配色倾向，不决定板块布局。
- **Page / Section Order**: Hero → 剧本开发 → 影视制作 → 教育培训 → 代表作品 → 样片展示 → 关于我们 → 联系我们 → 页脚，与需求 1:1 对齐。
- **Standard Content Zone**: Landing 页 `max-w-6xl` + `mx-auto`，保证桌面端大气舒展。
- **Shell / Frame Alignment**: 固定顶部导航与内容区同宽对齐，导航背景随滚动由透明过渡到深红实底。
- **Padding & Rhythm**: `px-4 md:px-6 lg:px-8`，板块垂直间距 `py-16 md:py-24`，保持 8px 倍数节奏。
- **Full-bleed Zones**: Hero 首屏、代表作品滚动海报区、页脚为全宽；Hero 内部文字与 CTA 受内容区约束。
- **Local Narrowing**: 联系我们板块二维码区域可收窄至 `max-w-3xl` 居中。
- **Overflow Strategy**: 影视制作 6 步时间轴、代表作品海报滚动、办公环境图片横向排列使用 `overflow-x-auto` 或 CSS 动画滚动。
- **Flexibility Boundary**: 允许移动端 padding、卡片列数、海报展示数量微调；不允许切换主色、圆角系统或金色装饰语言。

## 6. 视觉与动效

- **装饰**: 祥云纹低透明度底纹、金色细线分隔、龙形剪影首屏顶部、胶片齿孔暗纹、板块标题旁的中式印章/落款元素。
- **阴影/边界**: 中 —— 卡片带柔和深色投影 + 极细金色描边；按钮有金色微光外发光；图片有深色内阴影增加厚重感。
- **动效**: 精致 —— 平滑滚动（`scroll-behavior: smooth`）；导航栏滚动时背景透明度渐变；板块入场时从下方轻微上浮 + 渐显；金色元素 hover 时有微光流动效果；海报无限滚动使用 CSS `animation` 线性匀速。

## 7. 组件原则

- 按钮：主按钮朱红底 + 金色文字 + 金色细边框 + 微光 hover；次按钮透明底 + 金色边框 + 金色文字，hover 填充淡金。
- 卡片：暖米白底 + 极细金色描边 + 深色投影，标题用深朱红或金色，正文墨黑。
- 导航：固定顶部，初始透明（文字金色），滚动后深红实底（文字金色/米白）；当前锚点用金色下划线标识。
- 时间轴：金色节点 + 金色连接线 + 米白内容卡片，步骤编号用金色大号衬线字体。
- 所有交互元素必须有 `:focus-visible` 金色外发光状态。

## 8. Image Direction

- **Image Role**: Hero 背景图 + 板块装饰图 + 作品海报 + 办公环境图 + 样片缩略图。
- **Image Art Direction**: 整体走"东方影视美学"路线。Hero 背景以深红为底，顶部盘旋金色龙纹剪影，穿插祥云与电影胶片/摄影机暗纹，光线从右上方斜射，形成金色光束穿透感，材质像丝绸与金箔交织，情绪庄重而有力量。作品海报风格统一为电影海报质感，高对比、强光影、人物/场景主体突出，每张主色调中融入红金元素。办公环境图走现代中式商务风，木质家具 + 暖光 + 影视元素装饰，干净专业有文化底蕴。
- **Image Prompt Keywords**: 中国风龙纹祥云, 深红鎏金配色, 电影胶片暗纹, 丝绸金箔质感, 东方影视美学, 光束穿透效果, 庄重仪式感, 电影海报级光影, 现代中式商务空间, 高对比暗调氛围
- **Image Avoidance**: 避免卡通Q版龙、廉价素材图库感商务人物、过度饱和的正红正黄、塑料感金属光泽、无主题的抽象渐变、西式奇幻龙形象。

## 9. Anti-patterns

- **Split personality**: 板块之间各自为政，有的国风有的现代SaaS；全站统一红金系统与装饰语言。
- **Gold rush**: 金色滥用铺满标题、边框、图标、背景、按钮；金色只做 accent，控制在 10% 以内，用朱红与中性色托举。
- **Default SaaS drift**: 回到默认蓝按钮、白色卡片、无意义圆角大卡片；坚守朱金影韵的国风商务定位。
- **Invisible interaction**: hover 做了微光，focus-visible 丢了；每个可交互元素都要有金色外发光的键盘可见状态。
- **Mono-hue tyranny**: 朱红同时用于按钮、标题、图标、边框、链接；primary 只给 CTA 与品牌锚点，标题用金色，链接用金色下划线。
- **Cheap ornament**: 祥云龙纹到处贴，像廉价年画；装饰元素只出现在 Hero、板块标题旁、页脚等关键节点，低透明度、克制使用。
